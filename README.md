# Web Shepherd

A fun, interactive web app that demonstrates how shepherds guide herds of animals using simple rules.

## What is it?

This is a real-time visualization of a shepherding system. Watch as a group of agents (shepherds) autonomously work together to move a herd toward your chosen target. Click anywhere on the canvas to set a new goal—the shepherds will intelligently position themselves to guide the herd there.

## How it works

The simulation is based on agent-based modeling where each agent follows simple local behaviors:

**Herds** respond to nearby neighbors by:
- Staying separated (avoiding crowding)
- Moving in similar directions
- Staying attracted to the group
- Running away from shepherds

**Shepherds** navigate by:
- Moving toward a target location
- Pushing the herd forward
- Avoiding bumping into each other

Despite these simple rules, complex group behavior emerges—the herd naturally moves toward goals without explicit commands to individual members.

## Controls

- **Click the canvas** to move the target
- **Cursor toggle** — control a herd member or a shepherd directly with your mouse
- **Ranges toggle** — visualize the interaction zones around agents
- **Sizes** — adjust population sizes
- **Herd Behavior** — tune how herd members interact with each other
- **Shepherd Control** — adjust how shepherds guide the herd
